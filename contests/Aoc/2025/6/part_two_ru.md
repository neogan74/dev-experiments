# Часть 2

Головоногие признались, что не объяснили, как *читать* их примеры. Задачи по-прежнему разделены столбцом пробелов и используют оператор в нижней строке, но числа записаны иначе.

## Новые правила
- Математика пишется справа налево, по одному числу на столбец.
- Внутри столбца цифры идут сверху вниз: сверху — старшие разряды, снизу — младшие.
- Читайте столбцы справа налево, собирайте числа, затем применяйте оператор блока (`+` или `*`).
- Ответ листка — сумма результатов всех блоков.

## Пример
```
123 328  51 64 
 45 64  387 23 
  6 98  215 314
*   +   *   +  
```

Чтение по столбцам справа налево:
- Правая задача: `4 + 431 + 623 = 1058`
- Вторая справа: `175 * 581 * 32 = 3253600`
- Третья справа: `8 + 248 + 369 = 625`
- Левая: `356 * 24 * 1 = 8544`

Итог: `1058 + 3253600 + 625 + 8544 = 3263827`.

Решите листок заново по этим правилам. Какова общая сумма?
