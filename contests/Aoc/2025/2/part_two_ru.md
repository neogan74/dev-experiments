# Часть 2

Клерк обнаружил ещё недействительные ID. Похоже, юный Эльф использовал и другие паттерны.

Теперь ID недействителен, если он состоит из последовательности цифр, повторённой **как минимум дважды** (без ведущих нулей). Примеры: `12341234` (`1234` ×2), `123123123` (`123` ×3), `1212121212` (`12` ×5), `1111111` (`1` ×7).

На тех же диапазонах из примера:

- `11-22` всё ещё содержит два недействительных ID: `11`, `22`.
- `95-115` теперь содержит два недействительных ID: `99`, `111`.
- `998-1012` теперь содержит два недействительных ID: `999`, `1010`.
- `1188511880-1188511890` всё ещё содержит один недействительный ID: `1188511885`.
- `222220-222224` всё ещё содержит один недействительный ID: `222222`.
- `1698522-1698528` всё ещё не содержит недействительных ID.
- `446443-446449` всё ещё содержит один недействительный ID: `446446`.
- `38593856-38593862` всё ещё содержит один недействительный ID: `38593859`.
- `565653-565659` теперь содержит один недействительный ID: `565656`.
- `824824821-824824827` теперь содержит один недействительный ID: `824824824`.
- `2121212118-2121212124` теперь содержит один недействительный ID: `2121212121`.

Сумма недействительных ID в этом примере — `4174379265`.
