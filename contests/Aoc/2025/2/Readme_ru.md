# День 2: Сувенирная лавка

Вы выходите в сувенирную лавку — единственную другую остановку лифта. Весёлая вывеска благодарит за визит на Северный полюс. Отсюда можно пройти в вестибюль, а из него — в остальные части базы.

Клерк-эльф просит помощи: юный Эльф добавил в базу много недействительных ID товаров. Большинство диапазонов уже проверено; вам осталось обработать оставшиеся (ваш входной файл).

## Формат входных данных

- Диапазоны разделены запятыми.
- Каждый диапазон задаётся как `начало-конец` (включительно).

Пример диапазонов (перенесены для удобства чтения):

```
11-22,95-115,998-1012,1188511880-1188511890,222220-222224,
1698522-1698528,446443-446449,38593856-38593862,565653-565659,
824824821-824824827,2121212118-2121212124
```

## Правило недействительного ID

Недействительный ID состоит из последовательности цифр, повторённой дважды (без ведущих нулей). Примеры: `55` (`5` дважды), `6464` (`64` дважды), `123123` (`123` дважды). `0101` не встречается, потому что ID не начинается с нуля.

## Пример

- `11-22` содержит два недействительных ID: `11`, `22`.
- `95-115` содержит один недействительный ID: `99`.
- `998-1012` содержит один недействительный ID: `1010`.
- `1188511880-1188511890` содержит один недействительный ID: `1188511885`.
- `222220-222224` содержит один недействительный ID: `222222`.
- `1698522-1698528` не содержит недействительных ID.
- `446443-446449` содержит один недействительный ID: `446446`.
- `38593856-38593862` содержит один недействительный ID: `38593859`.
- В остальных диапазонах недействительных ID нет.

Сумма всех недействительных ID в примере равна `1227775554`.

## Задача

Найдите все недействительные ID во входных диапазонах и выведите их сумму.
