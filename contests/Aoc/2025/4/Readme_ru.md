## День 4: Печатный отдел

Вы поднимаетесь по эскалатору в печатный отдел. Здесь явно готовятся к Рождеству: повсюду огромные рулоны бумаги и массивный принтер в углу. Вам нужен путь дальше в базу Северного полюса, пока лифты не работают, и эльфы думают, что смогут помочь, если у их погрузчиков будет свободное время, чтобы пробить стену.

Если оптимизировать их работу, время появится.

### Задача
- Рулоны бумаги обозначены символом `@` на большой сетке (ваши входные данные).
- Рулон доступен погрузчику, только если в восьми соседних клетках находится **меньше четырёх** рулонов.
- Цель: определить, сколько рулонов доступны.

### Пример входных данных
```
..@@.@@@@.
@@@.@.@.@@
@@@@@.@.@@
@.@@@@..@.
@@.@@@@.@@
.@@@@@@@.@
.@.@.@.@@@
@.@@@.@@@@
.@@@@@@@@.
@.@.@@@.@.
```

### Пример вывода
Доступные рулоны отмечены символом `x` (всего 13):
```
..xx.xx@x.
x@@.@.@.@@
@@@@@.x.@@
@.@@@@..@.
x@.@@@@.@x
.@@@@@@@.@
.@.@.@.@@@
x.@@@.@@@@
.@@@@@@@@.
x.x.@@@.x.
```

### Итог
По полному плану размещения рулонов посчитайте, сколько из них доступны для погрузчика.
