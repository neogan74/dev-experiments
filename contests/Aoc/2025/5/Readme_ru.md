# День 5: Столовая

Погрузчики пробили стену, и эльфы с радостью нашли столовую по ту сторону. На кухне суета: нужно успеть украсить зал, а новый инвентаризационный учет только мешает.

## Задача
База данных работает с идентификаторами ингредиентов. Сначала идут диапазоны свежих ID, затем (после пустой строки) список имеющихся ID. Идентификатор считается свежим, если он попадает хотя бы в один диапазон.

## Формат входных данных
- Один или несколько диапазонов свежих ID в виде `начало-конец` (границы включены). Диапазоны могут пересекаться.
- Пустая строка-разделитель.
- Список имеющихся ID, по одному в строке.

## Пример
```
3-5
10-14
16-20
12-18

1
5
8
11
17
32
```

Разбор:
- 1: испорчен (не входит ни в один диапазон)
- 5: свежий (в диапазоне 3-5)
- 8: испорчен
- 11: свежий (в диапазоне 10-14)
- 17: свежий (в диапазонах 16-20 и 12-18)
- 32: испорчен

В этом примере свежими оказываются 3 имеющихся идентификатора.

## Цель
Обработать файл базы данных новой системы и посчитать, сколько из имеющихся идентификаторов ингредиентов свежие.
