# День 6: Мусорный пресс

После слишком бодрого прыжка в мусоропровод вы попадаете в магнитно запертую камеру пресса. Пока семейство головоногих пытается открыть дверь, вас просят помочь с математическим листком.

Каждый вертикальный блок столбцов в листке — это отдельная задача: верхние строки содержат числа, а нижняя строка хранит оператор (`+` или `*`). Задачи расположены рядом и разделены как минимум одним столбцом, полностью заполненным пробелами.

## Ввод
- Прямоугольная сетка из ASCII-символов.
- Последняя строка содержит операторы для каждой задачи.
- Столбцы, полностью состоящие из пробелов, разделяют задачи.
- Внутри блока задачи выравнивание чисел по левому/правому краю не важно.

## Вывод
Выведите общую сумму: вычислите каждую задачу с её оператором и просуммируйте все полученные результаты.

## Пример
```
123 328  51 64 
 45 64  387 23 
  6 98  215 314
*   +   *   +  
```

Задачи и ответы:
- `123 * 45 * 6 = 33210`
- `328 + 64 + 98 = 490`
- `51 * 387 * 215 = 4243455`
- `64 + 23 + 314 = 401`

Общая сумма: `33210 + 490 + 4243455 + 401 = 4277556`.
