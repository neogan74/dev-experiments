# День 3: Холл

Вы спускаетесь по короткой лестнице в огромный холл, проходите контроль безопасности и обнаруживаете, что все лифты выключены. Эльф у панели управления говорит, что всему виной всплеск напряжения; пока она чинит лифты, вы можете попробовать запитать эскалатор с помощью ближайших батареек.

Каждая батарейка помечена номиналом от `1` до `9`. Батарейки сгруппированы в банки, каждая строка цифр — это один банк. Внутри банка нужно включить ровно две батарейки; напряжение банка — двузначное число из выбранных батареек **в их текущем порядке**.

## Пример входных данных

```
987654321111111
811111111111119
234234234234278
818181911112111
```

## Как найти максимум банка

- `987654321111111` → лучше всего взять первые две батарейки: `98`.
- `811111111111119` → взять `8` и `9`, получится `89`.
- `234234234234278` → взять две последние, получим `78`.
- `818181911112111` → выбрать `9` и `2`, получится `92`.

Сумма этих максимумов даёт общее напряжение `98 + 89 + 78 + 92 = 357`.

## Задача

Для ваших входных данных вычислите наибольшее возможное напряжение для каждого банка и выведите общее напряжение.
