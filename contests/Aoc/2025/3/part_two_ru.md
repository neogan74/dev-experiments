# Часть Вторая

Эскалатор всё ещё не едет. Эльф много раз нажимает большую красную кнопку «снятие ограничения по напряжению», подтверждая каждый раз «да, уверена», пока вы украшаете холл.

Теперь в каждом банке нужно включить **ровно двенадцать** батареек. Напряжение банка — это 12-значное число из выбранных батареек **в их исходном порядке**.

## Пример входных данных

```
987654321111111
811111111111119
234234234234278
818181911112111
```

## Максимумы банков с 12 батарейками

- `987654321111111` → оставить всё, кроме пары завершающих `1`, чтобы получить `987654321111`.
- `811111111111119` → оставить всё, кроме части `1`, получив `811111111119`.
- `234234234234278` → убрать `2`, `3` и ещё одну раннюю `2`, чтобы получить `434234234278`.
- `818181911112111` → пропустить несколько начальных `1`, чтобы получить `888911112111`.

Общее напряжение: `987654321111 + 811111111119 + 434234234278 + 888911112111 = 3121910778619`.

## Задача

Для ваших входных данных выберите двенадцать батареек в каждом банке так, чтобы максимизировать напряжение банка, и выведите общую сумму.
